Quick Start

Commands to get the app running locally (replace names with your project values).

Install

npm install
# or
yarn install

Environment

PORT=3000
NODE_ENV=development
# Add DB_URL, JWT_SECRET, etc. to .env

Run

npm run dev
# or
node ./bin/www
# or if using nodemon
npx nodemon ./index.js

Health check

curl http://localhost:3000/api/health
# {"status":"ok","uptime":...}

API Reference

Common endpoints used by frontend and tests.

GET
/api/health
Returns service health and uptime
200 OK
GET
/api/users
List users (supports pagination)
200 OK
POST
/api/auth/login
Authenticate and return JWT token
200 OK / 401

Example: fetch token (frontend)

const resp = await fetch('/api/auth/login',{
    method:'POST',
    headers:{'Content-Type':'application/json'},
    body:JSON.stringify({email:'you@example.com',password:'secret'})
});
const json = await resp.json();
const token = json.token; // store in memory or cookie
